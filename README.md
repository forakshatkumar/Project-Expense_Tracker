# 💰 Expense Tracker

## About the Project

This Expense Tracker is a beginner-friendly web application built using **HTML, CSS, and JavaScript**. I created this project to strengthen my core JavaScript skills while building something practical that people can use to manage their daily income and expenses.

The application allows users to add and remove transactions, automatically updates the total balance, and displays spending distribution through an interactive **Chart.js** pie chart. To improve the user experience, transaction data is stored using **Local Storage**, ensuring it remains available even after refreshing the page.

## ✨ Features

- Add and delete income or expense transactions
- Automatic balance calculation
- Interactive expense visualization with Chart.js
- Persistent data using Local Storage
- Responsive and clean user interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Local Storage API
